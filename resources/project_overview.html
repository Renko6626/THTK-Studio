项目整体说明

概览

这是一个面向东方 Project 资源的脚本编辑器，支持 ANM/STD/MSG/ECL 多脚本类型，集成了解包/打包、语法高亮、结构大纲、悬浮说明与帮助面板等功能。

快速入门

  1. 通过菜单“文件 → 打开…”载入脚本文本（或在各面板的“解包”功能中选择源文件）。
  2. 右侧“帮助”面板支持悬浮查看指令说明；顶部下拉框可直接检索指令名。
  3. ECL 面板提供“结构大纲”Dock，可过滤/展开/双击跳转函数与标签。
  4. 完成编辑后使用相应面板的“打包”功能导出。

面板与功能

  • 帮助面板：显示光标下指令的签名与说明，支持下拉检索。
  • ECL 工具面板：设置 thecl 与 eclmap 路径、解包/打包、结构大纲。
  • 代码补全：根据已加载的参考 JSON 提供关键字与指令补全。
  • 语法检查：STD 支持分号检查；其他类型按需扩展。

路径与参考数据

应用会优先使用“用户路径设置”，否则回退到 resources/ 下的内置文件。

  • thecl.exe / thstd.exe / thmsg.exe 等工具
  • thecl_ref.json、thstd_ref.json 等指令说明
  • eclmap 等映射/类型定义

提示

  • 切换脚本类型时，相关 Dock 会自动加载/清理，避免界面堆积。
  • 若帮助文档未出现，请检查对应 JSON 路径设置。
